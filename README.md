This is a Task Manager API built with Node.js, designed for managing tasks efficiently. It includes file handling capabilities and utilizes the `formidable` package for file uploads. This project serves as a practice exercise for my course and demonstrates essential features like user authentication and image uploads.

## Features

- **Basic Task Management**: Supports Create, Read, Update, and Delete (CRUD) operations for tasks.
- **File Upload Functionality**: Enables users to upload files using the `formidable` package.
- **Node HTTP Server**: A simple HTTP server that manages API requests.
- **Authentication**: Basic user authentication to secure API endpoints.
- **Image Handling**: Upload and manage images, although currently, uploaded images do not retain their original filenames.
